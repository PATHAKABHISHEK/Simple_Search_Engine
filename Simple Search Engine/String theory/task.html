Description

Let's implement the simplest search engine possible. It searches for a specific word in a line containing various words.

The first line contains several words separated by a space. The words are numbered in order, with the first word having index 1. Suppose that all the words in the first line are unique, so there can be no repetitions.

Write a simple program that reads a single line, and then another line. The program must search in the second line for a word specified in the first line. The program should output the index of the specified word. If the word does not appear in the second line, the program should print "Not Found". Remember that indexes start from 1!

You should output exactly one line.

Examples

Example 1

Input:

first second third fourth
third

Output:

3

Example 2

Input:

cat dog and mouse
elephant

Output:

Not found
